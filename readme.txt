Skeleton plugin for G52GRP Software Engineering Group Project,
(University of Nottingahm) group gp11-cmg, intended to become a 
"learning dashboard" for moodle, i.e. an integrated and supportive
overview of a students entire programme of study, what they have
done so far, what is outstanding and what is future.

Chris Greenhalgh, University of Nottingham 2011

The site manager by default can view other user's user dashboards
(capability block/gp11cmg:viewuser)
